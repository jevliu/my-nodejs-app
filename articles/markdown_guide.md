 # Markdown 常用语法速查

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 粗体、斜体

```markdown
**粗体**
*斜体*
```

## 列表

```markdown
- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
```

## 链接与图片

```markdown
[百度](https://www.baidu.com)

![图片alt](https://www.example.com/image.png)
```

## 代码块

```markdown
`行内代码`

```
多行代码块
console.log('Hello Markdown!');
```
```

## 引用

```markdown
> 这是引用内容
```

## 分割线

```markdown
---
```

## 表格

```markdown
| 姓名 | 年龄 |
| ---- | ---- |
| 张三 | 18   |
| 李四 | 20   |
```

---

> 你可以在本博客系统的 .md 文件中自由使用以上语法，实时预览效果！
